LINUX CHEATSHEET FOR COMMANDS-
ls- lists in directories
pwd- present working directory
cd- change directory
cd ..- exit directory
mkdir- make directory
touch- creates an empty file
mv filename direct/ - moves file
cp filename direct/ - copy files
sudo apt install- install something
sudo apt update- shows available updates
sudo apt upgrade- actually update
ls -R - shows direct within direct
touch.filename- creates hidden file
ls -a - shows hidden files
top- shows processes using most resources
ps- shows running process
ps -a - shows background process
kill- kills the process via PID
ls -l - shows permissions
